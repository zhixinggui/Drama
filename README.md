# Drama
用來觀看 [8Drama][1] 影片的 Universal App.

![](README/1.png)

## 需求
* Xcode 7.1
* iOS 9.1


## 特色
* 使用 iOS 內建播放器, 但是支援手勢 (左滑後退 10 秒, 右滑前進 10 秒).
* Highlight 看過或是加入書籤的連結 (黃色).



[1]: http://8drama.com "8Drama"
